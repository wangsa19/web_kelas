## How to Use

1.Run git clone <my-cool-project>

    git clone https://github.com/wangsa19/web_kelas

2.Run

    composer install

3.Run

    cp .env.example .env

4 . Run

    php artisan storage:link

4.Run

    php artisan key:generate

5.Run

    php artisan migrate:fresh --seed

6.Run npm install

    npm install

7.Run

    php artisan serve

8.Go to link localhost:8000
